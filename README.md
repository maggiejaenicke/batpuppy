# Batpuppy
**Author:** Maggie Jaenicke

## Steps to serve locally

#### Clone the repo:
- HTTPS: `git clone https://github.com/maggiejaenicke/batpuppy.git`
- SSH: `git@github.com:maggiejaenicke/batpuppy.git`

#### Install dependencies: 
- `npm install` within project folder

### Local server
- `npm start` 

## Development workflow
- Create a branch off of `main`
- Stage changes: `git add .`
- Commit changes: `git commit -m "descriptive message"`
- Push changes to GitHub: `git push origin branchname`
- To merge changes into `main`, first make sure it is up to date (`git pull origin main`), then open a **Pull Request**
